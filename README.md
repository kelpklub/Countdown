# Countdown
A [Countdown website](https://kelpklub.github.io/Countdown/) that displays upcoming rocket launches in landscape mode.  
<img width="1366" height="768" alt="image" src="https://github.com/user-attachments/assets/c35c2595-68ce-4e2b-b55e-b0192161bd27" />

# Features  
- Realtime countdown.
- Cycles to next launch after liftoff.
- fullscreen toggle.
# Tech Stack  
- HTML5
- CSS3
- JavaScript
- HTML5 Canvas
- Launch Library 2 API
